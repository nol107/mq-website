# Backlinks & off-site authority — Maison Quasar

Google ranks pages partly on how many other reputable sites link to you.
On-site SEO can only do so much; **backlinks are how a small site escapes
obscurity**. Below: the targets that matter, in priority order.

---

## Tier 1 — Music platforms (do this first, takes 1 hour total)

These are profiles you control. Each one accepts a website link in the bio
or "About" section. Free, instant, high-quality backlinks.

- [ ] **Spotify for Artists** — bio + website field
- [ ] **YouTube channel "About" tab** — add maisonquasar.fr as the official link
- [ ] **SoundCloud** — bio + website
- [ ] **TikTok** — bio link (only one allowed; consider a Linktree)
- [ ] **Apple Music for Artists** — claim profile, add bio + website
- [ ] **Deezer for Creators** — same
- [ ] **Bandcamp** — if you ever release there

## Tier 2 — Event aggregators (per event)

Each upcoming event = a new entry on each of these. Set up artist profiles
once, then add events as they're announced. They link back to your site.

- [ ] **Shotgun** — artist profile + each event
- [ ] **Resident Advisor** — artist + events (RA is influential for electro)
- [ ] **Songkick** — artist profile (auto-syncs to Spotify, Bandsintown)
- [ ] **Bandsintown for Artists** — same
- [ ] **Le Bonbon Paris** — submit events via their form
- [ ] **Sortir à Paris** — same
- [ ] **TimeOut Paris** — long-shot but free to submit

## Tier 3 — French rap / hip-hop / electro media

Pitch them when you release a session or an EP. Use the email templates below.

- [ ] **Booska-P** — biggest French rap media. Hard to crack but worth trying.
- [ ] **Mouv'** (Radio France) — they have an "artistes émergents" segment
- [ ] **Konbini Music** — covers underground regularly
- [ ] **Le Bon Son**
- [ ] **Rap Tactics**
- [ ] **Hip Hop Corner**
- [ ] **L'ABCDR du Son** — long-form, niche but credible
- [ ] **Trax Magazine** — for electro/club nights
- [ ] **Tsugi** — same
- [ ] **Generations** (Skyrock group) — radio/web

## Tier 4 — Local Paris listings

Mostly for the studio + local SEO (see `LOCAL_SEO.md`).

- [ ] Pages Jaunes
- [ ] Paris.fr cultural directory
- [ ] Yelp Paris
- [ ] Mappy

## Email templates (FR, short, no fluff)

### Template A — Pitch a session video to a music blog

> Sujet : Quasar Session #5 — Ozaleji (rap français, Paris)
>
> Bonjour [Prénom],
>
> Je suis Nelson, du collectif Maison Quasar (Paris 19e). On vient de sortir
> notre 5e Quasar Session, avec **Ozaleji** — rap français, ambiance posée,
> tournée et produite intégralement chez nous.
>
> Lien : https://maisonquasar.fr/journal/quasar-session-5-ozaleji.html
> Vidéo : https://www.youtube.com/watch?v=CTvsDidp0JI
>
> Si ça vous parle, je serais ravi de vous en dire plus. Toutes les Quasar
> Sessions sont auto-produites au studio (beat → clip), c'est notre format
> récurrent.
>
> Bonne journée,
> Nelson — Maison Quasar
> https://maisonquasar.fr

### Template C — Ask a featured artist to backlink

> Hey [Prénom],
>
> J'espère que ça va. La page de ta Quasar Session est en ligne ici :
> https://maisonquasar.fr/journal/[slug].html

> tu veux qu'on mette quelque chose de spécifique en avant.
>
> Merci !
> Nelson

## Tracking

Keep a simple spreadsheet (Google Sheets is fine) with columns:

| Site | Type | Status | Date contacted | Date published | Backlink URL |

Track which outlets respond — most won't, but the ones that do become
ongoing relationships. After 6 months, look at Google Search Console
("Liens" report) to see which backlinks Google has indexed.
